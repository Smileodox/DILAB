# Regulatory Spectrum Monitoring - Rohde & Schwarz vendor perspective - source URL list

15 publicly available PDFs total (14 downloadable by `fetch.py`, 1 requiring a manual browser download).

Each entry: target filename (drop the file under `pdfs/<filename>`), URL.

> **Vendor-bias note.** This corpus is by definition vendor-biased. Use it to learn the *shape* of a complete monitoring system (antenna -> receiver -> DF/locator -> software -> network) and to extract concrete technology-driver candidates, but cross-check claims against the independent sources in sources.json (ITU, CEPT, FCC, NTIA, RSPG, academic surveys).
