Why this chain of methods

Every method has exactly one job

MethodIts jobWithout it
Morphological box (Zwicky 1969)opens the space — what is conceivableonly the obvious scenarios
Cross-impact panel (Weimer-Jehle 2006)closes it — what is consistent268 M arbitrary combinations
Monte-Carlo + fixed pointssearches it — every region, not cornershand-picked futures
HDBSCAN on ordinal encodingcondenses honestly — may say “no cluster”forced clusters
AHP + TOPSIS (Saaty; Hwang/Yoon)ranks transparentlygut feeling
Null-model refereeaudits every step against random fieldsunverifiable structure claims
None of these methods is new. New is that every hand-over between them is machine-executed and traceable — from archetype back to source paragraph.

Rigor

Test the metal detector before searching the beach

Field fed to the engineSilhouettez-scoreEngine’s verdict
Synthetic, coupled — a buried coin0.72+8.6beeps — usable structure
Synthetic, uncoupled — empty sand0.22−0.7silent — ≈ random
Real spectrum field (120 scenarios)0.07+1.4silent — ≈ random
z-score = how many standard deviations the structure sits above uniform-random fields of the same shape (20 draws). Same engine, same test — only the input differs.
The engine does not hallucinate clusters. The flat first reading of the real field was an honest measurement of the input — not a defect of the tool.

The finding

The field is a hybrid: five archetypes inside a continuum

Lens chain, same 120 scenarios: one-hot·k-means 0.07 → ordinal·k-means 0.17 → one-hot·HDBSCAN 0.34 (50 % noise) → ordinal·HDBSCAN 0.38 (33 % continuum)
⚠ 0.07 measures all 120 points; 0.38 only the dense core — different measurements, not an “improvement”. A third of the field honestly remains a continuum.
